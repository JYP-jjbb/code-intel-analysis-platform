# Other LLM Knowledge CSVs

This directory stores knowledge-base CSV files generated by non-primary LLM providers.

Planned files:
- `clknowledge.csv`: knowledge entries generated by CL source/model pipeline.
- `geknowledge.csv`: knowledge entries generated by GE source/model pipeline.
- `gpknowledge.csv`: knowledge entries generated by GP source/model pipeline.
- `grknowledge.csv`: knowledge entries generated by GR source/model pipeline.

Recommended future read entry:
- Backend service layer under Nutera/LLM domain, loading from classpath location:
  `data/knowledge_base/other_llms/*.csv`
- Keep CSV parsing and schema validation centralized in one loader component.
